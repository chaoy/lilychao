Hi, thanks for using this theme!

Pixel is a free template to use with the WordPress CMS. It features a 2-3 columns layout and sports the following goodies: dropdown top menu for categories; customizable feedburner field (optional, requires feedburner ID); widgets all around (6 positions); footer navigation and a transparent look. The divisions are also all made to fit standard ad formats and the welcome message on top right can also be customized through the admin panel, no need to mess into the code.

This theme, copyright sam @ http://85ideas.com, was designed and built for the WordPress CMS and is released under the GNU General Public License.
